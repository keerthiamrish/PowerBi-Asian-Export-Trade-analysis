# PowerBi-Asian-Export-Trade-analysis
Objective
This project analyzes export trade data using Power BI to identify trends,
country-wise performance, and export growth patterns,INR_USD growth.
Tools Used
- Power BI Desktop
- Excel
- Power Query
- DAX
 Dashboards
- Export Value Overview
- Country & Region Analysis
- Year-wise Export Trends
- Currency Impact Analysis
  Files
- Power BI report (.pbix)
- Dataset (Excel)
- Dashboard screenshots(pdf)
  Note
Live Power BI publish is not included due to account limitations.
The full Power BI report (.pbix) is provided for review.
